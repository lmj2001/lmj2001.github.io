配置虚拟环境

  1. 创建虚拟环境

    conda create --name 指明虚拟环境的名称 包名列表(多个以空格隔开)

  2. 激活和退出

  3. 导出虚拟环境的配置文件

    创建文件: conda env export > envconfig.yml

    是文件生效: conda env create -f envconfig.yml

    复制和克隆clone: conda env create [新的虚拟环境名称] --clone [老的虚拟环境名称]

  4. 删除虚拟环境
